Консольное приложение на Python, которое загружает исторические данные о курсе доллара США к российскому рублю с официального сайта Банка России (cbr.ru) и строит интерактивный график за указанный период.

## Возможности

- 📅 **Валидация дат** — проверка формата `DD/MM/YYYY`, логических ограничений (начало ≤ конец, не раньше 2000 года, не позже сегодняшнего дня)
- **Асинхронная загрузка** — получение XML-данных с ЦБ РФ через `aiohttp`
- **Красивый график** — построение диаграммы через `matplotlib` с:
  - форматированием оси Y в рублях (`₽`)
  - адаптивной сеткой и подписями
  - интерактивной подсказкой координат при наведении
- **Обработка ошибок** — понятные сообщения при некорректном вводе или проблемах с сетью

## Требования

- Python 3.10+
- Библиотеки: `aiohttp`, `matplotlib`
- Так же на линуксе требуется библиьтека для отрисовки окон tkinter установленая в систему 

## Использование

Запустите скрипт, передав начальную и конечную даты в формате `DD/MM/YYYY`:

```bash
python main.py 14/03/2001 20/06/2001
```
